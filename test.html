MAPeco - Test

Well, we could have said that the job was done, but actually how could we know if we did not ask anyone for their opinion. As the end of the semester was near and we did not have much time, we prepared a very short usability test.

The test containted three tasks:

  • 1) You would like to buy a coffee in your own cup, but the neighborhood is unfamiliar. What actions would you take?
  • 2) The 'Cleaning up the forest' event turned out to be a false initiative. Therefore, you would like to report that event. What would you do?
  • 3) You liked this location very much. Save the place for later and add a comment, please.

It turned out that not everything is clear for our users and thus there was some work ahead of us. As for the main view: we changed 'Criteria' tab to filter icon and changed the look of the icons on the map. And as for the filtering: we added an option to set the range of the searsh and another option for sorting the search results, changed the names of the categories and moved the option to plan an action from the main view.
